1) Ajuste de mapeo en el ETL (lo esencial)
En tu ETL (donde parseás cada fila de “Costos directos e indirectos”), aplicá estos mapeos y derivaciones:
// helper seguro: toma number si ya viene como number, si viene string usa parse flotante con puntos
const asNum = (v: any) => (v == null ? null : Number(v));

const hoursTarget   = asNum(row['Cantidad de horas objetivo']);
const hoursAsana    = asNum(row['Cantidad de horas reales Asana']);
const hoursBilling  = asNum(row['Cantidad de horas para facturación']); // preferido para costo
const rateARS       = asNum(row['Valor Hora']);        // ARS
const fx            = asNum(row['Cotización']);        // FX del mes (NO buscar "Tipo de cambio")
const totalUSDsheet = asNum(row['Monto Total USD']);   // ya viene calculado en la hoja

// Reglas de horas (como definimos):
const billH = (hoursBilling ?? hoursAsana ?? hoursTarget ?? 0);

// Derivaciones coherentes:
const costARS = (rateARS != null ? billH * rateARS : null);

// Si falta rate pero hay totalUSD y fx → retrocalculo rateARS
const rateFromUsd = (rateARS == null && totalUSDsheet != null && fx)
  ? (totalUSDsheet * fx) / (billH || 1)
  : null;

const finalRateARS = rateARS ?? rateFromUsd ?? 0;

// costUSD: preferir el de la hoja si existe; si no, derivar
const costUSD = (totalUSDsheet != null)
  ? totalUSDsheet
  : (costARS != null && fx ? costARS / fx : null);
Puntos clave del fix
