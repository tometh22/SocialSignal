Ese espacio gris es, casi con seguridad, el fondo de tu <body> asomando por debajo del header. Para “pegar” ese topbar celeste al tope de la ventana y seguir puliendo toda la UI, haz lo siguiente:
1. Quitar margen/padding global

En tu CSS base (reset o global), asegúrate de poner:
html, body {
  margin: 0;
  padding: 0;
  background: #F7F8FA; /* el gris clarito de fondo */
}
Así te quedará el body limpito y sin “franjas” sobrantes.
2. Hacer el header realmente sticky/full-width

Aplica estas reglas al contenedor de la barra azul:
.header {
  position: sticky;
  top: 0;
  left: 0;
  right: 0;
  height: 64px;
  z-index: 100;        /* por encima de sidebar */
  background: linear-gradient(90deg, #3B82F6, #2563EB); /* opcional: degradado sutil */
}
sticky en lugar de margin-tops arbitrarios.
left/right 0 garantiza que abarque sidebar + contenido.
3. Integración con la sidebar

Dale a tu layout un grid fijo donde el header ocupe toda la primera fila:
.layout {
  display: grid;
  grid-template-rows: 64px 1fr;
  grid-template-columns: 280px 1fr;
  height: 100vh;
}
.header  { grid-column: 1 / span 2; }
.sidebar { grid-row: 2; }
.main    { grid-row: 2; }
Así el azul queda “detrás” de la sidebar y no hay cortes.
Siguientes pasos para pulir la interfaz

Profundiza tu sistema de elevaciones
Define tokens de sombra (elevation-1, elevation-2…) y aplícalos homogéneamente a cards, modales y footer.
Revisa tu paleta y accesibilidad
Comprueba contraste de texto contra fondo (mín. 4.5:1).
Usa un único azul primario y variantes (hover, active, disabled) definidos en variables CSS.
Alineación en la cuadrícula de 8 px
Todo padding/margin: múltiplos de 8.
Inputs de 48 px alto, botones de 40–48 px, gutters de 24/32 px.
Unificar estilo de bordes
Border-radius de 8 px en todos los componentes redondeados.
Borde de inputs y cards: 1px solid #E0E0E0.
Microinteracciones coherentes
Hovers: cambio de sombra o fondo en 150 ms.
Focus: animar borde en 100 ms.
Transiciones de paso a paso (wizard): fade + slide de 200 ms.
Mejorar la navegación
Sidebar colapsable (solo iconos) en tablet/phone.
Añadir “hover tooltips” para cada ítem.
Marcar sección activa con barra lateral de 4 px en azul primario.
Detalles de contenido
Añade un pequeño breadcrumb sobre el form (“Cotizaciones > Nueva”) para orientación.
Incluye un buscador rápido de clientes arriba del combo “Seleccionar cliente”.
Refinar tipografía y jerarquía
Limita a 2 pesos (400, 600).
Escala de tamaños: 20 px (sección), 16 px (labels), 14 px (texto secundario), 12 px (hints).
