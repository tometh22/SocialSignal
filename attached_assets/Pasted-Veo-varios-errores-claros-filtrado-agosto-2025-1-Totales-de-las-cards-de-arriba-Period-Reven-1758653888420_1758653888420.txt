Normalizador de moneda robusto

Una única función para todo:

const toUSD = (raw: string|number|null) => {
  if (raw == null) return 0;
  const s = String(raw).replace(/\s/g,'')
    .replace(/[.$]/g,'')       // quita miles tipo “1.234.567”
    .replace(',', '.');        // coma decimal → punto
  const n = Number(s.replace(/[^\d.-]/g,''));
  return Number.isFinite(n) ? n : 0;
};


Úsala siempre para Monto_USD (ventas) y Total USD (costos).
