Acá tenés el reemplazo listo-para-pegar para que el Dashboard Ejecutivo use fact_cost_month como fuente de costos (y deje de tomar fact_labor_month).

1) Endpoint: /api/dashboard/executive-summary
a) Versión Drizzle (TypeScript)
// server/routes/executive-summary.ts
import { Router } from "express";
import { db } from "@/db";
import { sql, and, eq } from "drizzle-orm";
import {
  fact_rc_month,
  fact_cost_month,
  fact_labor_month,
} from "@/db/schema"; // ajusta paths

const router = Router();

/**
 * GET /api/dashboard/executive-summary?period=YYYY-MM
 * Ej: /api/dashboard/executive-summary?period=2025-09
 */
router.get("/api/dashboard/executive-summary", async (req, res) => {
  const period = (req.query.period as string) ?? ""; // "YYYY-MM"

  if (!/^\d{4}-\d{2}$/.test(period)) {
    return res.status(400).json({ error: "period debe ser YYYY-MM" });
  }

  // --- FINANCIERA (facturación y costos) ---
  // Costos salen de fact_cost_month (columna R del Excel, 1:1)
  // Facturación sale de fact_rc_month
  const [finance] = await db
    .select({
      periodKey: sql<string>`${fact_rc_month.period_key}`,
      revenueUsd: sql<number>`COALESCE(SUM(${fact_rc_month.revenue_usd}), 0)`.mapWith(Number),
      // 👇 clave: ahora el costo viene de fact_cost_month
      costUsd: sql<number>`COALESCE(SUM(${fact_cost_month.amount_usd}), 0)`.mapWith(Number),
    })
    .from(fact_rc_month)
    .leftJoin(
      fact_cost_month,
      and(
        eq(fact_cost_month.period_key, fact_rc_month.period_key),
        // (opcional) misma currency si manejaras múltiples; si no, remove
        // eq(fact_cost_month.currency, sql`'USD'`)
        sql`1=1`
      )
    )
    .where(eq(fact_rc_month.period_key, period));

  const marginUsd = finance.revenueUsd - finance.costUsd;
  const marginPct =
    finance.revenueUsd > 0 ? (marginUsd / finance.revenueUsd) * 100 : 0;

  // --- OPERATIVA (horas, personas activas, proyectos con actividad) ---
  const [ops] = await db
    .select({
      hoursAsana: sql<number>`COALESCE(SUM(${fact_labor_month.asana_hours}), 0)`.mapWith(Number),
      hoursBilling: sql<number>`COALESCE(SUM(${fact_labor_month.billing_hours}), 0)`.mapWith(Number),
      peopleActive: sql<number>`COUNT(DISTINCT ${fact_labor_month.person_key})`.mapWith(Number),
      projectsActive: sql<number>`COUNT(DISTINCT ${fact_labor_month.project_key})`.mapWith(Number),
    })
    .from(fact_labor_month)
    .where(eq(fact_labor_month.period_key, period));

  const billablePct =
    ops.hoursAsana > 0 ? (ops.hoursBilling / ops.hoursAsana) * 100 : 0;

  return res.json({
    period,
    financiera: {
      facturadoUsd: finance.revenueUsd,
      costosDirectosUsd: finance.costUsd, // ← ahora matchea Excel col. R
      margenUsd: marginUsd,
      margenPct: Number(marginPct.toFixed(1)),
    },
    operativa: {
      horasTrabajadas: ops.hoursAsana,
      horasFacturablesPct: Number(billablePct.toFixed(1)),
      personasActivas: ops.peopleActive,
      proyectosConActividad: ops.projectsActive,
    },
  });
});

export default router;


Notas:

fact_cost_month.amount_usd debe ser el total mensual de la columna R del Excel (“Monto Total USD”), sumado sin filtrar por horas.

fact_labor_month se mantiene solo para métricas operativas (horas, personas, proyectos).

b) SQL “helper” (opcional) como VIEW

Si querés simplificar el endpoint, podés crear una view y consultarla:

CREATE OR REPLACE VIEW executive_finance_month AS
SELECT
  r.period_key,
  COALESCE(SUM(r.revenue_usd), 0) AS revenue_usd,
  COALESCE(SUM(c.amount_usd), 0) AS cost_usd,
  COALESCE(SUM(r.revenue_usd), 0) - COALESCE(SUM(c.amount_usd), 0) AS margin_usd
FROM fact_rc_month r
LEFT JOIN fact_cost_month c
  ON c.period_key = r.period_key
GROUP BY r.period_key;


Luego en el endpoint solo haces:

const [row] = await db.execute(sql`
  SELECT period_key, revenue_usd, cost_usd, margin_usd
  FROM executive_finance_month
  WHERE period_key = ${period}
`);

Si el dashboard lee de project_aggregates, cambiá el ETL que la rellena para que total_cost_usd use fact_cost_month:

// ETL aggregation (pseudo)
const agg = await db
  .select({
    periodKey: fact_rc_month.period_key,
    revenueUsd: sql<number>`COALESCE(SUM(${fact_rc_month.revenue_usd}),0)`.mapWith(Number),
    costUsd: sql<number>`COALESCE(SUM(${fact_cost_month.amount_usd}),0)`.mapWith(Number), // 👈
  })
  .from(fact_rc_month)
  .leftJoin(
    fact_cost_month,
    eq(fact_cost_month.period_key, fact_rc_month.period_key)
  )
  .where(eq(fact_rc_month.period_key, period))
  .groupBy(fact_rc_month.period_key);
