2) Cómo calcular cada card (backend)
Usando Drizzle (Postgres) y period_key:
a) Facturación del período (USD)
const [{ revenueUsd }] = await db
  .select({
    revenueUsd: sql<number>`coalesce(sum(${factRcMonth.revenueUsd}),0)`.mapWith(Number)
  })
  .from(factRcMonth)
  .where(and(eq(factRcMonth.orgId, orgId), eq(factRcMonth.periodKey, periodKey)));
b) Ganancia del período (USD)
const [{ costUsd }] = await db
  .select({
    costUsd: sql<number>`coalesce(sum(${factRcMonth.costUsd}),0)`.mapWith(Number)
  })
  .from(factRcMonth)
  .where(and(eq(factRcMonth.orgId, orgId), eq(factRcMonth.periodKey, periodKey)));

const periodProfitUsd = (revenueUsd ?? 0) - (costUsd ?? 0);
Si en esa hoja algunos costos están solo en ARS, convertí a USD con el fx ponderado del mismo período (ver “fx” abajo) o con fx_rate de cada fila.
c) Horas del período
const [{ periodHours }] = await db
  .select({
    periodHours: sql<number>`coalesce(sum(${factLaborMonth.asanaHours}),0)`.mapWith(Number)
  })
  .from(factLaborMonth)
  .where(and(eq(factLaborMonth.orgId, orgId), eq(factLaborMonth.periodKey, periodKey)));
Si querés billables/non-billables: usá filter (where ...) con is_billable.
d) Proyectos activos (active/total)
Regla robusta (actividad en el mes o estado operativo):
const [{ active }] = await db.execute(sql<number>`
  select count(distinct p.id)::int as active
  from projects p
  left join fact_labor_month fl on fl.project_id = p.id and fl.period_key = ${periodKey}
  left join fact_rc_month fr    on fr.project_id = p.id and fr.period_key = ${periodKey}
  where p.org_id = ${orgId}
    and (
      p.status in ('ongoing','in_delivery')
      or coalesce(fl.asana_hours,0) > 0
      or coalesce(fr.revenue_usd,0) > 0
      or coalesce(fr.cost_usd,0) > 0
    )
`);

const [{ total }] = await db.execute(sql<number>`
  select count(*)::int as total from projects where org_id=${orgId}
`);
e) Tipo de cambio (ref.)
No promedio simple; sí ponderado por monto del período:
const [{ fxWeighted }] = await db
  .select({
    fxWeighted: sql<number>`
      coalesce(
        sum(${factRcMonth.revenueArs} + ${factRcMonth.costArs})
        / nullif(sum(${factRcMonth.revenueUsd} + ${factRcMonth.costUsd}), 0)
      )
    `.mapWith(Number)
  })
  .from(factRcMonth)
  .where(and(eq(factRcMonth.orgId, orgId), eq(factRcMonth.periodKey, periodKey)));
Si querés por cliente/proyecto, añadí group by.
Si manejás distintos fx_type, filtrá fx_type='operational'.
