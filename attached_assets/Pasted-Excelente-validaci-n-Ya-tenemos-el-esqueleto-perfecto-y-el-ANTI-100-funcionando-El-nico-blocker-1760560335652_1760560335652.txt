Qué pasa
La hoja Costos directos e indirectos trae Valor hora ARS = 0 para todos → al derivar cost_ars = billing_hours * hourly_rate_ars, queda 0.
Tus flags derived_cost_ars/usd confirman que no vino Total ARS en la fila y que derivamos por rate.
Cierre en 3 capas (recomendado)
1) Fuente (Excel) — corrección preferida
Completar “Valor hora ARS” por persona/proyecto/mes.
(Opcional) Si tenés Total ARS calculado en la hoja, completar esa columna; el ETL la prioriza sobre el rate.
Luego: re-ejecutar ETL y listo.
Si no podés corregir la hoja ya, seguí con 2) y 3) para que el sistema no se quede sin costos.
2) Catálogo de tarifas (persistente, por período)
Crear una tabla de soporte para tarifas y usarla como fallback si el Excel trae 0 o vacío.

Reglas de toma de tarifa (prioridad):
rate_excel (si > 0)
rate_catalog(person_id, project_id, period)
rate_catalog(person_id, null, period)
Última tarifa conocida del mismo person_id (mes anterior, histórico)
Tarifa base por rol (role_baseline_rate_ars)
Si nada: marcar flag rate_missing y (ver capa 3) usar tarifa provisional
3) Salvaguarda ETL (provisional para no cortar KPIs)
Cuando haya horas>0 y no tengamos ni Total ARS ni rate válido:
Opción A (provisional): usar última tarifa conocida del person_id (misma persona, cualquier proyecto) del mes anterior.
Opción B (rol): usar tarifa baseline del rol.
Opción C (reconciliación con RC): sólo si A y B no existen, repartir el RC.cost_native del período proporcional a las billing_hours. Marcar flag fuerte reconciled_to_rc_cost.
Así nunca queda Σcostos=0 si hay horas cargadas, y el Dashboard no se desconfigura. Queda auditado por flags y se puede ajustar luego con el catálogo.
Cambios mínimos en el ETL (pseudocódigo)
// Antes:
const rateARS = parseNum(row['Valor Hora ARS']);
const costARS = totalARSs || (billing * rateARS);

// Propuesto (prioridades/fallbacks)
let rateARS = parseNum(row['Valor Hora ARS']);

if (!(rateARS > 0)) {
  rateARS = lookupRateCatalog(person_id, project_id, period_key) ??
            lookupRateCatalog(person_id, null, period_key) ??
            lastKnownRate(person_id) ??
            roleBaselineRate(role_name) ?? 0;
}

let costARS = totalARSs; // si vino Total ARS de hoja, priorizarlo
let flags = [];

if (!costARS) {
  if (rateARS > 0) {
    costARS = billing * rateARS;
    flags.push('derived_cost_ars_from_rate');
  } else {
    // Salvaguarda (solo si hay horas)
    if (billing > 0) {
      const fallback = tryReconcileToRCCost(project_id, period_key, billing);
      if (fallback != null) {
        costARS = fallback;
        flags.push('reconciled_to_rc_cost'); // flag fuerte para auditoría
      } else {
        costARS = 0;
        flags.push('rate_missing_zero_cost');
      }
    } else {
      costARS = 0;
    }
  }
}

const costUSD = totalUSDs || (fx > 0 ? costARS / fx : 0);
