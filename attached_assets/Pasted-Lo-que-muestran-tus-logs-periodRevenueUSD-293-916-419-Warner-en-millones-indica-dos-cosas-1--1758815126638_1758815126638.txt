4) Regla anti-escala (anti ×100/×1000) en SoT — sin tocar Excel

Sumá una validación conservadora en el parser sólo para montos USD (fees mensuales razonables):

function desinflarSiEscala(valueUSD: number): number {
  // Simple y seguro: si termina en "000" y es > 200k, probar /100 y /1000
  if (valueUSD > 200000 && valueUSD % 1000 === 0) return valueUSD / 100;
  if (valueUSD > 2000000 && valueUSD % 1000 === 0) return valueUSD / 1000;
  return valueUSD;
}
