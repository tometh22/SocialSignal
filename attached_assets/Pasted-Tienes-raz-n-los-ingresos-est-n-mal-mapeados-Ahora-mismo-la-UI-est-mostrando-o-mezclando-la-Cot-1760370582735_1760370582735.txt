Tienes razón: los ingresos están mal mapeados. Ahora mismo la UI está mostrando (o mezclando) la Cotización como si fueran Ingresos. En tu hoja Rendimiento Cliente la fuente correcta de ingresos no es “Cotización”: son las columnas Facturación (USD) y Facturación [ARS]. Aquí va el cierre, corto y quirúrgico, para que Replit lo arregle sin dudas.
Origen exacto (SoT) de INGRESOS por proyecto-mes
De la pestaña Rendimiento Cliente:
Si el proyecto factura en USD (clientes USD):
Ingreso base = columna G: “Facturación (USD)”.
Si el proyecto factura en ARS (clientes ARS):
Ingreso base = columna I: “Facturación [ARS]”.
Nunca usar “Cotización” como ingresos. “Cotización” es el precio pactado (denominador del presupuesto), no la facturación del mes.
Cómo exponerlo en el payload (3 vistas)
Usa el mismo ingreso base y conviértelo sólo si la vista lo exige.
Vista Operativa (moneda del proyecto):
USD-nativo → views.operativa.revenue = Facturación (USD); currency='USD'
ARS-nativo → views.operativa.revenue = Facturación [ARS]; currency='ARS'
Vista USD consolidada:
views.usd.revenue = Facturación (USD) || (Facturación [ARS] / fx)
Vista Original:
views.original.revenue = Facturación (USD) || Facturación [ARS]
views.original.revenue_currency = 'USD' | 'ARS'
fx = tipo de cambio del período (columna Q o tabla de FX mensual).
Costos (para que no se crucen)
Costos [ARS] (columna K) = base local.
Costos (USD) (columna H o K/fx) = sólo si el proyecto es USD-nativo o si la vista es USD.
Cotización (denominador de Presupuesto)
Cotización (nativa) = columna F.
cotizacion_usd = F / fx (si el proyecto es USD-nativo o si la vista es USD).
% Presupuesto usado = costo de la vista / cotización de la vista (misma moneda).
Patch backend (pseudocódigo)
const isUSD = USD_CLIENTS.has(cliente);
const fx = fxByMonth[period];

const factUSD = num(rc['Facturación (USD)']);     // G
const factARS = num(rc['Facturación [ARS]']);     // I
const cotNat  = num(rc['Cotización']);            // F

const costARS = num(rc['Costos [ARS]']);          // K
const costUSD = num(rc['Costos (USD)']) || (fx ? costARS / fx : 0); // H o K/fx

// --- Vistas ---
const views = {
  operativa: isUSD
    ? { revenue: factUSD, cost: costUSD, currency: 'USD', cotizacion: cotNat / fx }
    : { revenue: factARS, cost: costARS, currency: 'ARS', cotizacion: cotNat },

  usd: { revenue: factUSD || (factARS / fx), cost: costUSD, currency: 'USD', cotizacion: cotNat / fx },

  original: {
    revenue: factUSD || factARS,
    revenue_currency: factUSD ? 'USD' : 'ARS',
    cost: costARS,
    cost_currency: 'ARS'
  }
};
