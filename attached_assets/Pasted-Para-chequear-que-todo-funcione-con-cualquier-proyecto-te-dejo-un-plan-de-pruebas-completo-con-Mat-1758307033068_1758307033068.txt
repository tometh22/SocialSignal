2) Invariantes a verificar (valen para cualquier proyecto)
A. Consistencia entre endpoints
/complete-data.summary.teamCostUSD = suma de actualCost en /deviation-analysis.deviations (±0.01).
/complete-data.summary.efficiencyPct = (ΣL / ΣK)*100 (si ΣK=0 → 70).
/complete-data.summary.revenueUSD = suma ingresos[].amountUSD.
markupUSD = revenueUSD - teamCostUSD (±0.01).
B. Por persona (deviation row)
hourDeviation = actualHours - budgetedHours.
deviationPercentage = K==0 ? 0 : ((L/K)-1)*100 (con signo).
severity respeta umbrales (por defecto 5/15/30).
C. Moneda
Todos los costos y presupuestos vienen en USD ya convertidos (la UI no divide por FX).
Ningún actualCost negativo.
D. Estados vacíos
Si no hay costos del período → summary.emptyStates.costos=true y no crashea.
Si no hay ingresos del período → summary.emptyStates.ingresos=true y no crashea.
E. Fuente
Equipo/Costos salen de “Costos directos e indirectos” (o tu BD local equivalente).
Ingresos salen de “Ventas Tomi” (o BD local equivalente).
No hay mezcla de fuentes.

3.2 Smoke por proyecto (ECON y EXEC)
while read -r PID; do
  echo "==== Proyecto $PID · $TF · ECON ===="
  SUM=$(curl -s "$BASE/api/projects/$PID/complete-data?timeFilter=$TF&basis=ECON" | jq '.summary')
  DEV=$(curl -s "$BASE/api/projects/$PID/deviation-analysis?timeFilter=$TF&basis=ECON")
  RANK=$(curl -s "$BASE/api/projects/$PID/performance-rankings?timeFilter=$TF" | jq '.summary // {}')

  echo "Summary:" && echo "$SUM"
  echo "Deviations length:" && echo "$DEV" | jq '.deviations | length'
  echo "TeamCost vs sum(actualCost):"
  TC=$(echo "$SUM" | jq -r '.teamCostUSD // .teamCost // 0')
  SUMAC=$(echo "$DEV" | jq '[.deviations[].actualCost] | add // 0')
  echo "  teamCostUSD=$TC  sumActualCosts=$SUMAC"
  # Invariante ±0.01
  python - <<PY
tc=$TC; sac=$SUMAC
assert abs(tc - sac) <= 0.02, f"TeamCostUSD mismatch: {tc} vs {sac}"
print("  ✔ invariant OK")
PY
  echo

  echo "==== Proyecto $PID · $TF · EXEC ===="
  curl -s "$BASE/api/projects/$PID/complete-data?timeFilter=$TF&basis=EXEC" | jq '.summary'
  curl -s "$BASE/api/projects/$PID/deviation-analysis?timeFilter=$TF&basis=EXEC" | jq '.summary, .deviations | length'
  echo
done < /tmp/projects.txt
