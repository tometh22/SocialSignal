1) Lectura correcta de la Columna R desde Google Sheets
Problemas típicos:
El range no incluye la col R.
Se usa valueRenderOption=FORMATTED_VALUE → llega texto con $, puntos y comas.
Fórmulas en R no se evalúan si no pedís el valor calculado.
Solución (Google Sheets API):
// googleSheetsWorking.ts
const range = `'Costos directos e indirectos'!A:R`; // asegurar hasta R

const resp = await sheets.spreadsheets.values.get({
  spreadsheetId,
  range,
  valueRenderOption: 'UNFORMATTED_VALUE',   // ← trae números crudos
  dateTimeRenderOption: 'FORMATTED_STRING',
});

const rows = resp.data.values ?? [];
// La cabecera debe tener "Monto Total USD" en index 17 (col R, base 0)
const header = rows[0].map(String);
const idxMontoUsd = header.findIndex(h => /monto\s*total\s*usd/i.test(h));
const idxMes      = header.findIndex(h => /^mes$/i.test(h));
const idxAnio     = header.findIndex(h => /^a(ñ|n)o$/i.test(h));
Si por alguna razón tenés que usar FORMATTED_VALUE, normalizá con:
Number(String(v).replace(/[^\d.-]/g,'')).
2) Parseo y mapeo robusto (sin filtrar por horas)
function toPeriodKey(mesStr: string, anioNum: number) {
  const map: Record<string,number> = {
    ene:1,feb:2,mar:3,abr:4,may:5,jun:6,jul:7,ago:8,sep:9,oct:10,nov:11,dic:12
  };
  const m3 = mesStr.slice(0,3).toLowerCase().normalize('NFD').replace(/\p{Diacritic}/gu,'');
  const mm = map[m3];  // ej "sep" -> 9
  return `${anioNum}-${String(mm).padStart(2,'0')}`;
}

for (let i=1; i<rows.length; i++) {
  const r = rows[i];
  const mes    = r[idxMes];         // ej "sep"
  const anio   = Number(r[idxAnio]); 
  const monto  = r[idxMontoUsd];    // con UNFORMATTED_VALUE llega Number | undefined

  const amountUsd = typeof monto === 'number' ? monto :
                    monto ? Number(String(monto).replace(/[^\d.-]/g,'')) : 0;

  if (!mes || !anio) continue;
  if (!amountUsd || amountUsd <= 0) continue;       // sólo descartá montos cero
  const periodKey = toPeriodKey(String(mes), anio);

  // upsert a una tabla dedicada de costos mensuales (recomendado)
  await db.insert(factCostMonth).values({
    orgId, periodKey, amountUsd, sourceRow: i, etlBatchId
  }).onConflictDoUpdate({
    target: [factCostMonth.orgId, factCostMonth.periodKey, factCostMonth.sourceRow],
    set: { amountUsd, etlBatchId, etlTs: sql`now()` }
  });
}
Clave: no uses ninguna condición de horas (Horas Asana > 0).
Esta tabla representa costos del mes tal cual Excel (Col R), independientemente de horas.
