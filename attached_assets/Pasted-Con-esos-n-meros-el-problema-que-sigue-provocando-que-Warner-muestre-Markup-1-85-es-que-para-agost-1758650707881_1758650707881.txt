Filtrar SIEMPRE por row.monthKey === monthKey antes de agregar.
C. Agregado por proyecto (un solo motor)
Para cada key y monthKey:
revenueUSD = Σ ingresos[key, monthKey]
costUSD    = Σ costos[key, monthKey]
workedH    = Σ horas_reales[key, monthKey]
targetH    = Σ horas_objetivo[key, monthKey]

profitUSD   = revenueUSD - costUSD
markupRatio = costUSD > 0 ? revenueUSD / costUSD : null     // mostrar “x.xx×”
marginPct   = revenueUSD > 0 ? profitUSD / revenueUSD : null // “xx.x%”
efficiency  = targetH > 0 ? workedH / targetH : null         // “xx.x%”
Estado activo
Fee → siempre activo.
One Shot → activo solo si revenueUSD || costUSD || workedH en el período.
